Scraping the URLs

Uses httr to fetch the HTML content of the downloaded URLs. The data is processed to extract text content from specific HTML elements.

stocks <- readRDS("stocks_.rds")
urls <- unlist(stocks)

library(httr)
df<- list()
for(j in 1:length(urls)){
  x <- GET(urls[j], add_headers('user-agent' = 'student scraping ([[your email]])'))
  
  df[[j]] <- x %>%
    read_html() %>%
    html_elements("span") %>%
    html_text()
}

Processing the text from the URLs

Cleans and structures the extracted text into a tidy format. Each document is assigned an extracted_name (a unique identifier), and the text is prepared for analysis.

Bigram Analysis

Tokenizes the text into bigrams (two-word phrases). Removes both generic stop words and custom stop words to focus on actionable phrases. Visualizes the top 20 bigrams in a bar plot.

# Define a custom list of stop words to exclude generic terms and date-related words
custom_stop_words <- c(
  "september", "30", "2024", "31", "2023", "fiscal", 
  "december", "form", "annual", "report", "form 10", "statement", "consolidated financial", "consolidated", "statements"
)

# Tokenize into bigrams, filter out custom stop words
bigram_df <- processed_df %>%
  unnest_tokens(bigram, df, token = "ngrams", n = 2) %>%
  separate(bigram, into = c("word1", "word2"), sep = " ") %>%
  filter(!is.na(word1), !is.na(word2)) %>%
  filter(!word1 %in% stop_words$word, !word2 %in% stop_words$word) %>%  # Exclude general stop words
  filter(!word1 %in% custom_stop_words, !word2 %in% custom_stop_words) %>%  # Exclude custom stop words
  count(word1, word2, sort = TRUE)

# Plot the top 20 actionable bigrams
bigram_df %>%
  slice_max(n, n = 20) %>%
  unite(bigram, word1, word2, sep = " ") %>%
  ggplot(aes(x = reorder(bigram, n), y = n)) +
  geom_bar(stat = "identity", fill = "steelblue") +
  coord_flip() +
  labs(
    title = "Top 20 Bigrams in Annual Reports",
    x = "Bigram",
    y = "Frequency"
  ) +
  theme_minimal()

Topic Modeling

Creates a Document-Term Matrix (DTM) from the processed data. Applies Latent Dirichlet Allocation (LDA) to identify 6 topics. Visualizes the top terms within each topic as a faceted bar plot.

# Create a document-term matrix
dtm <- analyze_df %>%
  count(extracted_name, word) %>%
  cast_dtm(extracted_name, word, n)

# Apply LDA with 4 topics
lda_model <- LDA(dtm, k = 4, control = list(seed = 1234))

# Extract topics
topics <- tidy(lda_model, matrix = "beta")

# Get top terms for each topic
top_terms <- topics %>%
  group_by(topic) %>%
  slice_max(beta, n = 10) %>%
  ungroup() %>%
  arrange(topic, desc(beta))

# Plot top terms per topic
top_terms %>%
  mutate(term = reorder_within(term, beta, topic)) %>%
  ggplot(aes(x = term, y = beta, fill = factor(topic))) +
  geom_bar(stat = "identity", show.legend = FALSE) +  # Hide legend if not needed
  coord_flip() +
  facet_wrap(~ topic, scales = "free", ncol = 2, strip.position = "top") +  # Adjust layout
  scale_x_reordered() +
  scale_y_continuous(labels = scales::percent_format(accuracy = 0.01)) +  # Format beta as percentage
  labs(
    title = "Top Terms in Each Topic",
    x = "Terms",
    y = "Beta (Proportion Contribution)"
  ) +
  theme_minimal() +
  theme(
    plot.title = element_text(hjust = 0.5, size = 16),
    axis.text.x = element_text(size = 7),  # Set x-axis text size
    axis.text.y = element_text(size = 7),  # Set y-axis text size
    strip.text = element_text(size = 10),  # Facet strip text size
    panel.spacing = unit(1.5, "lines")  # Add space between facets
  )

Sentiment Analysis Insights

Highest Sentiment Scores: hi-20240930, ghix-20231231, tve-20240930 exhibit the most positive sentiment in their annual reports.

Implications: These companies are likely emphasizing favorable financial results, growth opportunities, or overall stability. Stakeholders (e.g., investors, creditors) might perceive these companies as lower-risk, making them attractive for investment or lending.

Action: Analyze these reports further to understand the drivers of positive sentiment, such as significant revenue growth or successful investments. Consider these companies as benchmarks for understanding effective communication strategies in annual reports.

Lowest Sentiment Scores: talo-20231231, vrex-20240927, hp-20240930 show the most negative sentiment.

Implications: These companies may be facing operational or financial challenges, such as declining revenues, litigation risks, or unfavorable market conditions. Potential risks for stakeholders could include increased volatility or the need for restructuring.

Action: Investigate the specific sections of their reports contributing to the negative sentiment (e.g., “Risk Factors,” “Management Discussion and Analysis”). Monitor these companies closely for turnaround strategies or further declines.

Keyword Frequency Insights

Positive Words: Fair, Significant, Outstanding are the most frequent positive terms.

Implications: Companies are emphasizing fairness (likely in valuation or governance), significance (in decisions, acquisitions, or partnerships), and outstanding performance.

Action: Look for specific contexts where these words appear to understand how companies frame their successes. Ensure reports balance positive framing with realistic assessments to maintain credibility.

Negative Words: Debt, Risk, Loss are the most frequent negative terms.

Action: Identify sectors or companies where these terms dominate to evaluate whether risks are being actively managed or escalating. Advise companies to provide clear mitigation strategies alongside mentions of risks to reassure stakeholders.
